# Introduction to Linux

This course provides a comprehensive introduction to Linux systems, covering essential shell commands, filesystem management, networking fundamentals, and basic server security.

## Modules

The course is structured into sequential exercises and lectures:

- **Shell and Linux Intro**: Getting started with the command line interface.
- **Filesystems and Permissions**: Understanding Linux directory structures and user rights.
- **Disks Formatting**: Managing storage devices.
- **Networking Fundamentals**: Basic network configuration and troubleshooting.
- **Security and Server Setup**: Best practices for hardening Linux servers.
- **Final Exercise**: A comprehensive challenge to apply learned skills.

## Pedagogical Goal

The goal of this course is to provide students with a solid foundation for managing Linux systems securely and effectively. All exercises are designed to be hands-on and practical.
