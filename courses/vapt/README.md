# Vulnerability Assessment and Penetration Testing (VAPT)

This course covers the methodologies and tools used for vulnerability assessment and penetration testing in modern IT environments.

## Course Content

- **Introduction and Recap**: Foundations of security testing.
- **DVWA & Metasploitable**: Practical testing environments for exploring common web and infrastructure vulnerabilities.
- **Infrastructure Security**: Securing network devices and systems.
- **Software Security**: Identifying and mitigating software-level vulnerabilities (e.g., buffer overflows).
- **Final Challenges**: Practical capstone projects to test assessment skills.

## Ethical Warning

The techniques taught in this course involve testing for security weaknesses. **These techniques must only be used on systems that you own or have explicit permission to test.** Unauthorized testing is illegal and unethical.
