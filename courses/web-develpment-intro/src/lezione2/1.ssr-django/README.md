# Esempio SSR (Django)

Questo file `template.html` mostra la sintassi utilizzata da Django per generare pagine HTML dinamiche sul server.

## Concetti chiave
- `{{ variabile }}`: Iniezione di un valore semplice.
- `{% for ... %}`: Ciclo per generare elementi ripetuti (es. righe di una tabella o elementi di una lista).
- `{% if ... %}`: Logica condizionale per cambiare l'aspetto della pagina in base ai dati.
