# Esempio SSG (Hugo)

Questo file `.md` rappresenta un tipico contenuto per un generatore di siti statici come Hugo.

## Struttura
- **Frontmatter**: La parte racchiusa tra `---` contiene i metadati (titolo, data, tag) in formato YAML.
- **Content**: Il corpo del file è scritto in Markdown, che verrà convertito in HTML durante la fase di "build".
