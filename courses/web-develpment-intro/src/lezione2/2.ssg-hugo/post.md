# Perché usare Hugo?

Il **Web Statico** non è morto, anzi! Grazie a strumenti come **Hugo**, possiamo generare siti web incredibilmente veloci.

## Vantaggi di un SSG
1. **Velocità di Build**: Hugo compila migliaia di pagine in pochi secondi.
2. **Markdown**: Scrivi i contenuti in questo formato semplice (come questo file!).
3. **Temi**: Migliaia di layout pronti all'uso.
