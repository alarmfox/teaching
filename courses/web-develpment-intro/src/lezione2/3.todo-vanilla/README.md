# Todo App - Vanilla JS

Un'applicazione completa per la gestione dei task che mette in pratica i concetti di logica Client-Side.

## Funzionalità (CRUD)
- **Create**: Aggiunta di nuovi task tramite l'input.
- **Read**: Visualizzazione dinamica della lista.
- **Update**: Possibilità di segnare un task come completato.
- **Delete**: Rimozione dei task.

## Persistenza
I dati vengono salvati nel `localStorage` del browser. Se ricarichi la pagina, i tuoi task saranno ancora lì!
