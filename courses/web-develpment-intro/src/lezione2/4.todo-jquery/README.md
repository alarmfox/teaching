# Todo App - jQuery Version

Versione della Todo App che utilizza la libreria jQuery per una manipolazione del DOM più sintetica e cross-browser.

## Differenze rispetto alla versione Vanilla
- Uso del selettore `$(...)`.
- Gestione degli eventi semplificata con `.click()`.
- Manipolazione dinamica della lista con `.empty()` e `.append()`.
- Inclusione della libreria tramite CDN nel file `index.html`.
