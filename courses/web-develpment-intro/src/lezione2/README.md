# Lezione 2: Il Web Statico vs Dinamico e la Logica Client-Side

In questa cartella sono presenti gli esempi di codice relativi alla seconda lezione del corso.

## Contenuti

1.  **1.ssr-django**: Esempio di Server-Side Rendering utilizzando il motore di templating di Django (Python).
2.  **2.ssg-hugo**: Esempio di contenuto per uno Static Site Generator (Hugo) in formato Markdown.
3.  **3.todo-vanilla**: Implementazione di una Todo App utilizzando esclusivamente HTML, CSS e JavaScript moderno (Vanilla JS). Focus su CRUD e localStorage.
4.  **4.todo-jquery**: La stessa Todo App implementata con l'ausilio della libreria jQuery per semplificare la manipolazione del DOM.

## Obiettivi Didattici
- Comprendere la differenza tra rendering lato server e lato client.
- Imparare a gestire lo stato dell'applicazione nel browser (persistenza locale).
- Praticare le operazioni CRUD (Create, Read, Update, Delete) su una struttura dati dinamica.
