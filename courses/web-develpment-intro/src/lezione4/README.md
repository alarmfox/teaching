# Lezione 4: Sviluppare il proprio Backend

In questa cartella troverete il materiale pratico per la lezione 4, dedicata alla creazione di un server backend e all'integrazione con il frontend.

## Struttura della cartella
- `1.basic-todo/`: Il punto di partenza per l'esercitazione. Contiene un backend che gestisce i dati inizialmente in memoria (volatili).
- `2.complete-solution/`: La soluzione completa con gestione della persistenza manuale, feedback di caricamento (spinner) e simulazione di latenza.

## Requisiti
Per eseguire gli esempi, assicurati di avere installato:
- **Node.js** (per la versione Express)
- **Python 3** (per la versione Flask)
