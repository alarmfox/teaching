# DTLab Challenge: Challenge Day

## Overview
This module contains the materials for the DTLab Challenge Day. It is designed to test a variety of cybersecurity skills, ranging from network traffic analysis and credential harvesting to system exploitation and cryptographic recovery.

## Contents
- **`brd-plaintext-pwd/`**: Contains a Python script demonstrating insecure network credential broadcasting. This challenge focuses on traffic sniffing and protocol analysis.
- **`enc-dec/`**: Contains a comprehensive challenge requiring asset discovery, network access, service exploitation, and cryptographic decryption to recover a final flag.

## Ethical Usage Warning
These exercises are for educational purposes only. They involve intentionally insecure configurations and practices (such as cleartext transmission of credentials and vulnerable services). **Never deploy these configurations in a production or public-facing environment.**
