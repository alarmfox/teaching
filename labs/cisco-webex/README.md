# Cisco Webex Integration Lab

This lab consolidates previous project work (Gematica, MaticMind, Threat Intelligence Enrichment System) into a unified example demonstrating the use of Cisco Webex cards and integrations.

## Contents
- **Gematica**: Configuration files and infrastructure data.
- **MaticMind**: Application code and Docker setup for the bot and REST API.
- **Threat Intelligence Enrichment System**: Use case documentation and API testing tools.

## Goal
Integrate these components to show how they leverage Cisco Webex for threat intelligence notifications and automated responses.
